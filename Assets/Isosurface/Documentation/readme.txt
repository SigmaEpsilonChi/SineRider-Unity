----------------------------------------

Foreword
----------------------------------------
Thank you for purchasing Isosurface! This editor extension is a modeling program for creating and building meshes based on mathematical formulas. And all without leaving the Unity environment. If you have any comments or suggestions, please email me at support@meshmaker.com.

Notes
----------------------------------------

As a general rule any mathematical formula that is valid within c# (i.e. follows the Math class rules) should work within this program.
