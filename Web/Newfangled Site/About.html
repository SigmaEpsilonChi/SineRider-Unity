About SineRider

SineRider is a game inspired by the many hours I spent graphing on a TI-86.

Each level is a mathematical puzzle which can be solved by composing a function.

Some of the puzzles are very simple.

Others are more complex.

Puzzle solutions can be shared with links, like this one.

The project is in early alpha right now, but is still very playable. All feedback is welcome.
